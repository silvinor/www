{%- comment -%} Renders the metadata line (date, reading time, categories, tags) for a post. Usage: {% include post-meta.html post=post %} post : the post to render metadata for (required) {%- endcomment -%} {%- assign post = include.post -%} {%- assign reading_time = post.content | number_of_words | divided_by: 200 | at_least: 1 -%}
{{ post.date | date: site.date_format }} · {{ reading_time }} min read {%- if post.categories.size > 0 -%} · {%- for category in post.categories -%} {{- category -}}{%- unless forloop.last -%}, {% endunless -%} {%- endfor -%} {%- endif -%} {%- if post.tags.size > 0 -%} · {%- for tag in post.tags -%} {{ tag }} {%- endfor -%} {%- endif -%}
